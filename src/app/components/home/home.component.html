Help Desk Suelen Feijo - Programação Descomplicada

Olá! Seja bem vindo a um projeto baseado em um sistema de Help Desk.
Esse é um sistema para gerenciamento de chamados simulando um simples Help Desk desenvolvido em Angular 12 e Spring Boot 2.
O sistema está rodando na nuvem sendo o front na Vercel e o back na AWS.
O projeto tem como objetivo principal abordar os conceitos de aplicações SPAs, Autenticação e Autorização com Tokens JWT. O código fonte do projeto pode ser acessado clicando no link Github no menu lateral.
Brevemente o sistema contara com serviço de autenticação de dois fatores , serviços de troca de mensageria, e muito mais.

Fique a vontade para testar o sistema e deixar sua sugestão.
